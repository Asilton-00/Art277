Quick Glossary

HTML (HyperText Markup Language)
HTML is the standard language used to create and structure content on a web page.
It defines the layout and elements such as headings, paragraphs, links, images, tables, and forms.

CSS (Cascading Style Sheets)
CSS is used to style and design web pages controlling how HTML elements look on the screen.
It handles colors, fonts, layouts, spacing and animations.

JavaScript (JS)
JavaScript is a programming language that makes web pages interactive and dynamic.
It allows users to click buttons, fill forms, show pop-uos, or update content without reloading the page.
